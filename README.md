# Chappell Hill Wine & Spirits 🍷

This is the website for **Chappell Hill Wine & Spirits**, a local liquor store in Petal, Mississippi.

The goal of this project is simple:
- Make it easy to find the store
- Share hours, location, and photos
- Create a clean, modern look for a community business

---

## About the Site

This is a **frontend-first** website focused on design, layout, and user experience.  
It’s built to be simple now, with the option to grow later if needed (products, admin panel, or backend features).

---
